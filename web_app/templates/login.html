Railway Track Fault Detection using Region-Based Convolutional Neural Networks for People Safety

This project focuses on enhancing railway safety by using Artificial Intelligence to automatically monitor railway tracks and identify defects that may lead to accidents.

Project Goal

  • Develop an AI-based railway track monitoring system.
  • Use Region-Based CNN (R-CNN) for accurate fault localization.
  • Classify tracks as Safe or Defective.
  • Trigger alerts when defective tracks are detected.
  • Improve railway safety and prevent accidents.

Project Details

  • Deep learning model: Faster R-CNN.
  • Detects defective regions with bounding boxes.
  • Supports image and video-based detection.
  • Integrated alert mechanism for fault detection.
  • Decision Accountability: Each detection includes the decision result, confidence score, risk level, reasoning, and recommended action for transparent monitoring.
  • Web application developed using Flask.
